# SkyNotes ☁️ 

**SkyNotes** is an aesthetic, motion-driven productivity web application designed for a seamless and visually pleasing user experience. Built with a focus on soft glassmorphism and fluid animations, it combines sticky notes, a task manager, and a minimal calculator into one cohesive workspace.

---

## ✨ Features

* **Draggable Sticky Notes**: Create, edit, and move notes anywhere on the canvas. 
* **Smart Persistence**: Your notes, their positions (X, Y), and your task list are automatically saved to `localStorage`.
* **Glassmorphic UI**: A modern, "Apple-style" aesthetic using Tailwind CSS with soft sky-blue gradients and backdrop blurs.
* **Interactive To-Do List**: Quick task entry with toggleable completion states and slide-in animations.
* **Minimalist Calculator**: A floating widget for quick calculations without leaving your workspace.
* **GSAP Motion Design**: High-performance animations for element entrance, note dragging, and deletions.

## 🛠️ Tech Stack

* **HTML5**: Semantic structure and modern layout.
* **Tailwind CSS**: Utility-first styling for the glassmorphic aesthetic.
* **JavaScript (ES6+)**: Modular logic and state management.
* **GSAP (GreenSock)**: Professional-grade motion and drag-and-drop physics.
* **LocalStorage API**: Browser-based data persistence (no backend required).
